# Kifyie — Creative Dev & Interactive Designer

My personal portfolio — built to showcase my work, projects, and journey as a creative developer & interactive designer.

🌐 **Live:** [kifyie.pro](https://kifyie.pro)

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Build | Vite + TypeScript |
| Markup | HTML5 (single-page) |
| Styling | Vanilla CSS + Tailwind (utilities) |
| Animation | GSAP, Anime.js, CSS transitions |
| Scrolling | Lenis smooth scroll |
| Hosting | Vercel |

---

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
